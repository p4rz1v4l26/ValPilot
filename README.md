## YouTube

### uptime
```
/youtube/uptime?channelId=<channelid>
```

<br>

### Subscriber Count
<br>

###### With channelid

```
/youtube/subscribers?channel=<channelId>&useID=true
```
###### Without channelid
```
/youtube/subscribers?channel=<channelname>&useID=false
```
